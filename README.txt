Em cada uma das questões utilizei uma ferramenta diferente, para demonstrar um pouco da minha diversidade no que diz respeito ao meu aprendizado atual. Não busco aprender somente um framework ou uma linguagem, meu objetivo é ser um 
desenvolvedor fullstack o mais preparado possível. 
Questão 1:
Utilizei REACT, no momento em que eu recebi o desafio, estava exercitando e fazendo uma calculadora de IMC, aproveitei a tela para adequar a lógica ao que foi solicitado.
Para rodar abra um terminal na pasta e execute o comando npm install para instalar as dependências e o comando npm start para rodar a aplicação.  

Questão 2: 
Consegui desenvolver apenas com o vscode aberto utilizando Javascript, esta questão foi a que tive menos dificuldade para desenvolver. 
Para rodar basta abrir o arquivo index. 

Questão 3: 
Levei algum tempo para conseguir entender o que foi solicitado e adequar meus conhecimentos. Desenvolvi em Java utilizando o STS. 
Para rodar, basta abrir o java(sts, eclipse) e executar o código.
